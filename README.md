# 🤖 AI Student Assistant

An AI-powered student assistant built using **n8n**, **Groq**, and **Simple Memory**.

## 🚀 Features
- 💬 AI Chat Agent
- 🧠 Conversational Memory
- ⚡ Groq LLM Integration
- 🔄 n8n Workflow Automation

## 🛠️ Tech Stack
- n8n
- Groq
- AI Agent
- Simple Memory

## 📂 Workflow
Import the `student-assistant-ai-agent.json` workflow into n8n, configure your Groq API key, and start chatting.

## 📦 Installation
1. Download the workflow JSON.
2. Import it into n8n.
3. Configure your Groq API key.
4. Run the workflow.
